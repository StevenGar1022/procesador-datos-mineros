
# Procesador de Datos Mineros en Python
Este es un script script simple y eficiente en Python diseñado para automatizar la lectura y consolidación de datos de producción minera.

##  Características
* **Lectura automatizada:** Lee un archivo de texto (`minerales.txt`) con formato `Mineral - Cantidad`.
* **Procesamiento de datos:** Utiliza diccionarios para agrupar y sumar las cantidades de cada mineral de forma dinámica.
* **Generación de reportes:** Exporta un archivo limpio (`reporte_mineral.txt`) con los totales consolidados.

## Tecnologías utilizadas
* Python 3.x (Manejo de archivos, diccionarios y estructuras de control).
